# Uni-Reserve

## Room Booking and Event Management System

### Project Overview
Uni-Reserve is a DBMS-based University Reservation System designed to manage room bookings, event scheduling, and booking requests within a university environment.

### Features
- Room Booking Management
- Event Scheduling
- Individual Booking Requests
- Group Booking Requests
- User Authentication
- Group Management

### Technologies Used
- SQL
- DBMS
- MySQL

### Project Contents
- ER Diagram
- Relational Schema
- DDL Scripts
- INSERT Scripts
- SQL Queries and Solutions
